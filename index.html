Banner-boi

Banner Boi is an automated rapid development HTML5 Banner toolkit to help you get those pesky banners out the door quickly and efficiently. It's fully customisable and scaleable for current and future AD providers. Last but not least, it's EASY to use!

View on GitHub Download .zip Download .tar.gz

Features

  • Clean(I hope) & Easy to Use
  • Includes SASS, CSS Pleeease, GSAP/TweenLite/TimelineLite, and other goodies
  • Packaged provider support for DoubleClick, Sizmek, ... more to come soon
  • Fully customizable
